### Installation

requires [Node.js](https://nodejs.org/) v10+ to run.

Install the dependencies  and start the server.

```sh
$ npm install -d
$ node start
```


### Build
```sh
npm run build
```


### Run

requires [pm2](https://github.com/Unitech/pm2/blob/master/README.md)

```sh
pm2 start apps.json
```

### Stop

```sh
pm2 stop all
pm2 kill
```
